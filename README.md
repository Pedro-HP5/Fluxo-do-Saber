# 📚 Fluxo do Saber

O **Fluxo do Saber** é uma plataforma digital mobile-first no modelo marketplace voltada para o ambiente acadêmico de ensino superior. O objetivo central é promover a economia colaborativa, permitindo que estudantes comprem, vendam e troquem materiais educacionais de forma prática e segura.
