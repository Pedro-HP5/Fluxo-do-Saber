# 📚 Fluxo do Saber

O **Fluxo do Saber** é um marketplace acadêmico colaborativo focado na economia circular dentro de campus universitários. O projeto foi desenvolvido para facilitar a compra, venda e troca de materiais didáticos entre estudantes, visando a sustentabilidade e a redução de custos acadêmicos.

## 🔗 Navegação do Sistema
Devido à estratégia de prototipagem em módulos independentes para garantir alta fidelidade visual e funcional, o sistema está estruturado em fluxos de acesso direto:

* **[🔒 Módulo de Autenticação]** (Link para o Vercel do Login)

## 🛠️ Tecnologias Utilizadas
* **Gerador de Interface:** v0 (Vercel)
* **Engenharia de Prompt:** Gemini (Google)
* **Framework / Biblioteca:** React e Next.js
* **Estilização:** Tailwind CSS e Shadcn/UI
* **Ícones:** Lucide React
